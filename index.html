APOLO

Welcome! This is APOLO, a framework that will guide you step by step to design a Demand-Responsive Transport (DRT) system for a strategic-level project. First, you’ll choose the variables that are the main reasong behind the need for the existence of the DRT and that will guide the selection of the other elements. Then you’ll set elements that will define how the DRT will operate on a daily basis and simulate those in a virtual enviroment. Lastly, thorugh a cost/benefic analysis, you will define the elements that shape how the system and the passengers interact.

Move through the sections in order and confirm each choice—your selections are saved locally. At the end of each section, you can click on "PDF Report" to export the progress so far. Need changes? Use the BACK buttons to edit and generate a new PDF. That’s it—you’re ready to begin!
