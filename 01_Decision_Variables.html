Directive Variables

The Directive Variables assist the user in determining the primary objectives of the system. These variables will guide the entire development of the DRT, influencing decision-making and reflecting whether the selected characteristics align with the previously defined objectives. Subsequently, the user will be presented with the possible values for three decision variables: purpose, target passengers, and service hours. In the fourth decision variable, service area, the user specifies the geographical area in which the DRT will operate.
